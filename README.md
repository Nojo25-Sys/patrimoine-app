# 🏛️ Patrimoine App

Application de gestion des patrimoines avec FastAPI.

## Installation
```bash
git clone https://github.com/abotc/patrimoine-app.git
cd patrimoine-app
pip install -r requirements.txt
```

## Lancement
```bash
python -m uvicorn main:app --reload
```

## Accès

- Interface : http://127.0.0.1:8000/page/login
- Documentation API : http://127.0.0.1:8000/docs
